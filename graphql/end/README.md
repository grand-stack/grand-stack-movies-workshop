### Quick start guide

```bash
npm install
export NEO4J_URI=bolt://<your value here>:7687/
export NEO4J_USER=<your value here>
export NEO4J_PASSWORD=<your value here>
npm start
```

The username and password for the default provided dataset is neo4j/admin.
